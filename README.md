## How to run the program
1. Download the repository
```
git clone https://github.com/chrischapman1/eWoCCDatabaser
``` 
3. Open the files in Visual Studio
4. Set your MS SQL database URL
5. Run with C# compiler

## Files Required
- modelInputs.xml
- EventLogs (folder)
